## 2. Pipeline de ingesta de datos (ETL)

### 2.1 INEI — Precios de referencia

```
Programación: mensual (cuando INEI publica nuevo índice)

1. Verificar si hay nuevo .xlsx en inei.gob.pe
2. Si hay: descargar → parsear con openpyxl → upsert a tabla `precios_referencia`
3. Si no hay: mantener datos del mes anterior
4. Registrar versión/fecha de la tabla usada (RF-SCO-11)
```

### 2.2 SEACE / OCDS — Metadatos de obra

```
Programación: diaria (o bajo demanda)

1. Consultar API OCDS por rango de fechas o por entidad
2. Extraer: obra (título, monto, ubicación), contratista (RUC, razón social),
   entidad, fechas
3. Obtener coordenadas desde campo de ubicación o geocodificar
4. Upsert a tablas `obras`, `contratistas`, `contratos`
```

### 2.3 INFOBRAS — Partidas del expediente vía Gemini

```
Programación: bajo demanda (cuando se consulta una obra por primera vez)

1. Obtener URL del PDF del expediente desde INFOBRAS
2. Descargar PDF → convertir a base64
3. Enviar a Gemini API con prompt estructurado:
   "Extrae la tabla de partidas del siguiente expediente técnico.
    Devuelve un array JSON con objetos {insumo, unidad, cantidad,
    precio_unitario}. Usa estos nombres exactos."
4. Validar respuesta:
   - ¿Tiene los campos requeridos?
   - ¿Los precios están en rangos esperados? (ej. cemento > 0 y < 200)
   - Si validación OK → insertar en `partidas_obra`
   - Si no → marcar obra como "fallback" y usar RF-SCO-08
5. Almacenar respuesta cruda de Gemini en `log_extraccion` para auditoría
6. Cachear resultado para no reprocesar en consultas subsecuentes
```

### 2.4 JNE — Autoridades

```
Programación: semanal

1. Descargar dataset de autoridades electas/vigentes
   (datosabiertos.gob.pe — JNE)
2. Parsear CSV/XLSX → upsert a tabla `autoridades`
3. Vincular autoridades con municipio y período
```

## 4. Modelo de precios de referencia

### 4.1 INEI — Precios nacionales (fuente primaria)

Los **Índices Unificados de Precios de la Construcción (IUPC)** que publica el INEI mensualmente son **índices de precios** (variación porcentual respecto a un año base), no precios absolutos en S/. Adicionalmente, el INEI publica cuadros de **precios absolutos por insumo** (cemento, acero, agregados, mano de obra, etc.) a nivel **nacional**.

| Aspecto | Detalle |
|---|---|
| Cobertura geográfica | **Nacional** (no disponible por departamento para materiales individuales) |
| Formato | .xlsx descargable |
| Cadencia | Mensual |
| Contenido | Precios unitarios absolutos en S/ por insumo (cemento, fierro, madera, etc.) |

El motor de scoring compara el precio declarado en la partida contra el precio nacional INEI. En la UI se indica claramente: *"Precio de referencia: nacional (INEI)"* para que el usuario entienda el contexto.

### 4.2 Ministerio de Vivienda — Costo/m² por departamento (fallback regional)

El **Ministerio de Vivienda, Construcción y Saneamiento** publica anualmente los **Valores Unitarios Oficiales de Edificación** con costos por metro cuadrado de construcción **por departamento** (24 departamentos). Estos valores cubren:

- Costo por m² según tipo de edificación (vivienda, local comercial, etc.)
- Diferencias regionales por departamento
- Actualización anual

Se usan en el fallback RF-SCO-08 para validar el presupuesto total de la obra contra el costo regional esperado.

**Pipeline de ingesta:**

```
Programación: anual (enero, cuando el MVCS publica nuevos valores)

1. Descargar Resolución Ministerial con los Valores Unitarios Oficiales
2. Parsear tablas por departamento y tipo de edificación
3. Upsert a tabla `precios_referencia` con fuente = 'mvivienda'
```

### 4.4 Impacto en el cálculo del score

```
precio_referencia_a_usar = 
  IF existe precio INEI para el departamento:
      precio_inei_departamento      ← prioridad 1 (INEI regional si existe)
  ELSE IF existe precio INEI nacional:
      precio_inei_nacional           ← prioridad 2 (caso común)
  ELSE:
      NULL → partida marcada "no comparable" (RF-SCO-10)

ratio = precio_declarado / precio_referencia_a_usar

# El factor regional NO se aplica al ratio
# Se muestra solo como contexto en el desglose explicativo:
# "Este insumo cuesta 35% más que el promedio nacional.
#  En Loreto los costos de construcción son ~35% más altos
#  que en Lima (fuente: MVCS)."
```

### 4.5 Estrategia de pricing (resumen)

| Fuente | Tipo de dato | Cobertura | Uso en scoring |
|---|---|---|---|
| INEI — IUPC | Precios absolutos por insumo (S/) | **Nacional** | Score primario por partida |
| Ministerior Vivienda — VUO | Costo/m² por tipo de obra (S/) | **Por departamento** | Fallback RF-SCO-08 |
| Factores de ajuste | Factor adimensional | Estimado por departamento | Contexto visual en desglose |
